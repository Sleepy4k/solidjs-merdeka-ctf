## How to install

- Download it from link that given to you

- Install dependency

~~~bash
npm install
~~~

- Copy env file

~~~bash
copy .env.example .env
~~~

- Set env to storage or on file (don't reveal your key and env)

~~~bash
APP_DEBUG={category|(true,false)}
~~~

- Run program

~~~bash
npm start
~~~
